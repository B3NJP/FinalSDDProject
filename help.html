Basic Controls

The basic controls for Swordcraft are:
ESC - Open Menu
WASD - Move player UP, LEFT, DOWN, RIGHT respectively
Arrow Keys - Camera movement
UHJK - Use combo in direction UP, LEFT, DOWN, RIGHT respectively
M - Open stat menu
N - Open combo Menu
1-9 - Select combo 1-9 Note: Combos are displayed on the combo menu

Menus

Swordcraft has several menus:
A title screen, a pause menu and a combo creation menu
The title screen is self explanatory showing both a start new game and quit button
The pause menu has three options:
Resume - Continue playing and unpause
Combo Menu - Create new combos and edit existing ones
Quit - Ends the game

Combo Menu

The Combo Menu displays 4 columns:
Combos, Arts, Current Combo, Directions

The Combos column displays all the combos the player has created. At the bottom is a new combo button that creates a new combo. When creating a combo, the player has to name it by typing the name and pressing enter. The name is not displayed while typing.

The Arts column shows all the arts a player can use. It gets longer when more arts are learned. At the bottom, there are page buttons to change which page of arts is displayed. Clicking on one of the arts will add it to the currently selected combo.

To the right of the Arts column is the current combo column. This column shows the combo name at the top followed by the arts said combo consists of below. The arts are listed in the order they are used. Clicking on the name of the combo will rename it. Like typing the name for the first time, the name will not be shown until after pressing enter. Clicking on any of the arts in this column will remove the art from this combo. At the bottom is a delete combo button which will delete the selected combo.

Furthest to the right is the directions column. At the top of the directions column is a counter which displays how many points the combo costs vs the amount of points the player has. Combos and Arts

Combos and arts are the primary (and only way for the player) to attack in this game. A combo is a collection of arts that trigger them in sequence. Arts are the basic building blocks of combos and are used as basic attacks.

An art has a few primary features:

  • Arts have a modifier that affects how it will interact with an enemy. These include increasing physical attack to lowering the enemy's defence.
  • Arts will move the enemies around.
  • Arts will move the player around.
  • Arts may hit multiple spaces.
For instance, the basic art Pierce:
Attacks the one space in front, and moves the player and enemy one forwards.

The basic art Broad Slash:
Attacks three spaces in front and moves enemies to the right, but without moving the player
