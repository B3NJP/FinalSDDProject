Basic Controls

The basic controls for Swordcraft are:
ESC - Open Menu
WASD - Move player UP, LEFT, DOWN, RIGHT respectively
Arrow Keys - Camera movement
UHJK - Use combo in direction UP, LEFT, DOWN, RIGHT respectively
M - Open stat menu
N - Open combo Menu
1-9 - Select combo 1-9 Note: Combos are displayed on the combo menu

Menus

Swordcraft has several menus:
A title screen, a pause menu and a combo creation menu
The title screen is self explanatory showing both a start new game and quit button
The pause menu has three options:
Resume - Continue playing and unpause
Combo Menu - Create new combos and edit existing ones
Quit - Ends the game
